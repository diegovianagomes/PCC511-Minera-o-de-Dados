Tratamento de Valores Ausentes¶

  • Imputação de valores ausentes na coluna 'Age'.
  • As medianas de idade são calculadas para cada combinação de 'Pclass' e 'Title' a partir do conjunto de treino.
  • Valores de 'Age' ausentes são preenchidos com a mediana do grupo correspondente.
  • Se um grupo não tiver mediana (ex: combinação rara), usa a mediana global do treino como fallback.
  • O output mostra as medianas calculadas por grupo e confirma que não há mais NaNs em 'Age' após a imputação.

def impute_age_grouped(cols, medians):
    Age = cols.iloc[0]
    Pclass = cols.iloc[1]
    Title = cols.iloc[2]

    if pd.isnull(Age):
        try:
            return medians.loc[Pclass, Title]
        except KeyError:
            return train_df['Age'].median()
    else:
        return Age

Codificação de Variáveis Categóricas¶

  • Aplica One-Hot Encoding às features categóricas restantes (Pclass, Sex, Embarked, Title, Deck, AgeBin, FareBin).
  • Na aplicação One-Hot Encoding garatimos que apenas as colunas que existem em AMBOS e são categóricas são usadas para get_dummies para evitar erro.
  • Converte essas colunas em um formato numérico (colunas binárias) adequado para modelos.
  • A coluna 'Survived' é separada do conjunto de treino, tornando-se a variável alvo (y).
  • As colunas dos conjuntos de treino e teste são alinhadas para garantir que tenham o mesmo conjunto de features após a codificação.
